# Get team info

```txt copy filename="GET"
https://valorant-esports1.p.rapidapi.com/v1/teams/{teamid}
```

Provides detailed information about a Valorant team. By making a GET request to this URL with the `teamid` parameter, you will get information such as team name, players, staff, events, results and upcoming events in which the team participates.

## Headers

- `X-RapidAPI-Key`
- `X-RapidAPI-Host`

## Parameters

| Parameter | Default | Required | Description                    |
| :-------- | :-----: | :------: | :----------------------------- |
| `teamid`   |         |    âœ…    | ID of the team to be consulted |
