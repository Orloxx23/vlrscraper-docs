# Get player info

```txt copy filename="GET"
https://valorant-esports1.p.rapidapi.com/v1/players/{playerid}
```

Provides information about a specific Valorant Esports player. By making a GET request to this URL with the specified `playerid`, you will get details about their personal information, team they belong to, and links to their social networks.

## Headers

- `X-RapidAPI-Key`
- `X-RapidAPI-Host`

## Parameters

| Parameter  | Default | Required | Description                 |
| :--------- | :-----: | :------: | :-------------------------- |
